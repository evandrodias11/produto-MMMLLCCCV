Exercício: Leitura de Código

Suponha que uma fábrica utilize como código de produto, um número inteiro no formato MMMLLCCCV, onde:
• Os três primeiros dígitos, representados pela letra M, correspondem ao modelo do produto;
• Os dois dígitos seguintes, representados pela letra L, correspondem ao número do lote;
• Os três dígitos seguintes, representados pela letra C, correspondem ao código da cor (01-Vermelho; 02-Preto; 03-Amarelo);
• O último dígito, representado pela letra V, corresponde à voltagem (1 = 110v; 2=220v);

Crie uma página que leia o código do produto e imprima os dados do produto, conforme o exemplo:

Produto: 100020012
Modelo: 100
Lote: 2
Cor: Vermelho
Voltagem: 220v
